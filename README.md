# 🇩🇴 RD Quiz – Conoce Tu País

PWA educativa con preguntas sobre República Dominicana: provincias, municipios, presidentes, historia, leyes, geografía y más.

## 🔗 Jugar

👉 **[Abrir RD Quiz](https://tobipg.github.io/Proyecto-preguntas-RD/)**

Funciona en cualquier navegador (computadora o celular). En el celular puedes "Agregar a pantalla de inicio" para usarlo como app, incluso sin conexión.

## 🧪 Para los beta testers

Gracias por probar el juego. Algunas cosas a tener en cuenta:

- Tienes **5 vidas diarias** que se recargan automáticamente.
- Puedes activar **1 inmunidad gratis al día** (no pierdes vidas por 5 minutos).
- Ganas **monedas jugando** (1 moneda por cada 20 puntos), úsalas en la tienda para comprar vidas extra.
- En la tienda hay un campo para **canjear código de promoción**. Códigos disponibles para esta beta:
  - `BIENVENIDO`
  - `RDQUIZ2026`
  - `BETATESTER`

Si encuentras un error, una pregunta mal redactada, o algo que no funciona, avísanos con: el modo de juego, la pregunta exacta (si aplica) y qué esperabas que pasara.

## 🛠 Desarrollo

Proyecto estático (HTML/CSS/JS vanilla), sin build step.

```bash
# Ejecutar la suite de tests
npm test
```

Para probar localmente, sirve la carpeta con cualquier servidor estático, por ejemplo:

```bash
npx serve .
```
